# CoT-RAG LLM Gene Prioritization for Disease Diagnosis

This repository encompasses a framework designed to prioritize genes associated with diseases by leveraging large language models (LLMs) for reasoning and analysis.

## Description

Synthetic clinical notes are generated by us using GPT-4 from data at Phenopacket-Store. This ensures a high-quality representation of clinical data for our analyses.
